# Prompt Harbor

A clean, personal AI prompt library PWA. Collect, version, and manage your prompts — all stored locally in your browser.

## Features

- **Multi-Version Prompts** — Store and manage multiple versions of each prompt with custom version labels
- **Search & Filter** — Quickly find prompts by title, tags, model, or content
- **Import & Export** — Export your library as JSON and import it on any device
- **PWA Ready** — Install on your phone or desktop, works offline like a native app
- **Local Storage** — All data is stored locally in your browser. Nothing is uploaded to any server

## Tech Stack

- Pure HTML + CSS + JavaScript (no frameworks)
- localStorage for data persistence
- Service Worker for offline caching
- PWA manifest for installability

## Usage

Open `index.html` in a browser, or deploy to any static hosting:

- **GitHub Pages**: Push to `main` branch, enable in repo Settings → Pages
- **Cloudflare Pages**: Connect repo, set build output to `/` (root)
